O qué é CSS


Trata-se de uma linguagem de marcação, amplamente utilizada com HTML ou XHTML,
e representa diversas possibilidades para a formatação. O CSS ajuda a editar,
alinhar, remover e trabalhar no espaço entre elementos de uma página.

O qué é JS


JavaScript é uma linguagem de programação de alto nível criada, a princípio,
para ser executada em navegadores e manipular comportamentos de páginas web.

O qué é React

O React é a biblioteca mais popular do JavaScript e é usada para construir uma interface de usuário (IU).
Ela oferece uma resposta excelente para o usuário adicionar comandos usando um novo método de renderizar sites.

O qué é Java

Java é uma linguagem de programação e plataforma computacional lançada pela primeira vez pela Sun Microsystems em 1995.
Existem muitas aplicações e sites que não funcionarão,a menos que você tenha o Java instalado,
e mais desses são criados todos os dias. O Java é rápido, seguro e confiável
